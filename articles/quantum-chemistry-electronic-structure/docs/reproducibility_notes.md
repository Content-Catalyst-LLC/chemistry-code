# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible quantum chemistry workflows should preserve:

- molecular identifiers
- coordinates
- charge and spin multiplicity
- method
- basis set
- exchange-correlation functional when relevant
- dispersion correction when relevant
- solvation model when relevant
- convergence criteria
- grid settings
- starting geometry
- optimized geometry
- frequency calculation details
- imaginary frequencies
- energy components
- orbital outputs
- electron-density outputs
- software versions
- input files
- output files
- post-processing scripts
- benchmark references
- uncertainty and limitations
- provenance records

Future extensions could include:

- Psi4 input templates
- NWChem input templates
- orbital visualization metadata
- cube-file scaffolds
- geometry optimization logs
- frequency analysis summaries
- reaction-path scans
- excited-state tables
- transition-metal spin-state comparisons
- benchmark-dataset integration
