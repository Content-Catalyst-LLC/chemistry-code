# Full-Stack Computational Extensions

This folder extends the Chemistry, Classification, and the Human Understanding of Matter article repository with full-stack computational examples.

## Added languages

- Python — full synthetic chemical classification workflow and report generation
- SQL — provenance schema and classification query examples
- Julia — classification scoring and reliability modeling
- C — evidence, reliability, materiality, and mixture-complexity kernels
- C++ — class triage and class-level aggregation
- Fortran — feature-space similarity scenario modeling
- Rust — safe systems-style classification screening
- Go — lightweight class summaries
- R — statistical summaries for classification reliability and hazard indicators

## Responsible-use boundary

All examples are synthetic and educational. They do not identify real unknowns, validate laboratory results, determine regulatory classification, replace analytical chemistry, assign legally binding hazard categories, produce safety data sheets, certify purity, or substitute for expert review.
