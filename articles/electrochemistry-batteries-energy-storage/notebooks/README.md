# Notebook Notes

Suggested notebook scaffolds:

1. `electrochemistry_energy_storage_audit.ipynb`
   - imports synthetic cell, cycling, voltage, impedance, materials, and lifecycle tables;
   - calculates capacity, energy, retention, and coulombic efficiency;
   - ranks energy-storage candidates;
   - flags degradation, impedance, safety, and critical-material review items;
   - writes reproducibility outputs.

2. `electrochemistry_energy_storage_report.qmd`
   - renders a reproducible teaching report;
   - links figures and tables to generated outputs;
   - documents limitations and responsible-use boundaries.

Notebook files should be executed from a clean kernel to avoid hidden-state errors.
