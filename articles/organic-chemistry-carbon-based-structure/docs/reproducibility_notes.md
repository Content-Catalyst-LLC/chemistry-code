# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible organic-structure workflows should preserve:

- molecular formula
- atom identities
- bond connectivity
- bond orders
- stereochemical descriptors
- conformational assumptions
- functional group definitions
- protonation state
- tautomer assumptions
- aromaticity model
- units and descriptor definitions
- computational scripts
- output artifacts
- software versions
- provenance records
- limitations and responsible-use notes

Future extensions could include:

- RDKit-based descriptor generation
- SMILES and InChI parsing
- tautomer enumeration
- stereoisomer enumeration
- conformer generation
- substructure searching
- spectral metadata linking
- quantum chemistry inputs
- molecular property prediction
- reaction-template extraction
