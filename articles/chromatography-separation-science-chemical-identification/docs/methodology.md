# Methodology Notes

This folder models a reproducible chromatography workflow using synthetic educational data.

The workflow demonstrates:

1. retention-time and retention-factor calculations;
2. adjacent-peak resolution calculations;
3. tentative retention-time candidate matching;
4. calibration from synthetic peak-area standards;
5. unknown concentration estimation;
6. chromatographic method metadata;
7. full-stack scientific-computing scaffolding.

Retention-time matching is used only as a teaching example. Real chemical identification requires stronger evidence, including reference standards, spectral information, uncertainty estimates, method validation, and expert review.
