# Chromatography Reproducibility Checklist

- Preserve raw chromatograms and instrument-export files.
- Record method, instrument ID, column ID, carrier/mobile phase, and detector settings.
- Record temperature program, gradient program, flow rate, injection volume, and sample preparation.
- Document peak integration settings and manual interventions.
- Preserve standards, blanks, quality controls, and suitability checks.
- Distinguish tentative retention-time matches from confirmed identifications.
- Store processed data and figures with source provenance.
- Report uncertainty, limitations, and alternative interpretations.
