# Responsible Use Notes

This repository is educational.

Do not use these scripts for:

- authoritative reference data
- chemical safety decisions
- laboratory handling decisions
- regulatory compliance
- environmental reporting
- clinical chemistry
- pharmaceutical quality control
- industrial process control
- materials qualification
- isotope-ratio interpretation without expert review

Professional chemical practice requires validated data sources, trained personnel, calibrated instruments, documented assumptions, uncertainty analysis, safety procedures, and regulatory compliance where applicable.
