# Responsible Use Notes

This repository is educational.

Do not use these scripts for:

- clinical diagnosis
- regulatory reporting
- forensic evidence reporting
- pharmaceutical batch release
- environmental compliance decisions
- food safety decisions
- hazardous laboratory procedures
- validated laboratory information management
- safety-critical industrial process control

Professional analytical chemistry requires validated methods, calibrated instrumentation, trained personnel, documented quality systems, reference materials where appropriate, uncertainty analysis, and regulatory compliance where applicable.
