# Methodology Notes

## Purpose

This repository demonstrates reproducible computational scaffolding for analytical chemistry and the identification of matter.

## Core Relationships

Linear calibration:

S = m x + b

Unknown concentration:

x = (S - b) / m

Limit of detection:

LOD = 3 s_b / m

Limit of quantification:

LOQ = 10 s_b / m

Relative standard deviation:

RSD = 100 s / mean

Spike recovery:

Recovery = 100 (spiked - unspiked) / spike_added

Beer-Lambert law:

A = epsilon b c

Chromatographic resolution:

Rs = 2(tR2 - tR1) / (w1 + w2)

Signal-to-noise ratio:

SNR = signal / noise

## Scientific Caution

These examples use simplified educational assumptions. Real analytical chemistry requires validated methods, matrix assessment, calibration design, reference materials, uncertainty analysis, quality-control rules, trained personnel, and fit-for-purpose interpretation.
