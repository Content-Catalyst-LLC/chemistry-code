# Methodology Notes

## Purpose

This repository demonstrates reproducible computational scaffolding for introductory chemistry.

## Core Calculations

Amount of substance:

n = m / M

Molarity:

C = n / V

Dilution:

C1 V1 = C2 V2

First-order kinetics:

[A](t) = [A]0 exp(-kt)

pH:

pH = -log10([H+])

Beer-Lambert law:

A = epsilon * l * c

## Scientific Caution

These examples use simplified educational assumptions. Real laboratory chemistry requires safety review, calibration, uncertainty analysis, validated methods, proper standards, and domain expertise.
