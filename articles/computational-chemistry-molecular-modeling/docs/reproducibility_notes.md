# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible computational chemistry workflows should preserve:

- molecule identifiers
- structure representations
- coordinates
- charge and multiplicity
- method and basis set
- force-field parameters
- solvent model
- conformer-generation method
- geometry optimization settings
- frequency calculation settings
- molecular dynamics settings
- random seeds where relevant
- software versions
- input files
- output files
- post-processing scripts
- benchmark references
- uncertainty and limitations
- provenance records

Future extensions could include:

- RDKit descriptor workflows
- Psi4 input templates
- conformer generation with real molecular structures
- geometry optimization logs
- frequency scaling examples
- docking metadata templates
- molecular dynamics trajectory analysis
- reaction-path energy profiles
- descriptor-based machine learning
- materials and periodic-system scaffolds
