# Reproducibility Notes

All datasets are synthetic and educational.

Reproducible chemistry-history workflows should preserve:

- input values
- units
- reaction assumptions
- mass-balance assumptions
- nomenclature mappings
- scripts
- output artifacts
- software versions
- provenance records
- limitations and responsible-use notes

Future extensions could include:

- scanned historical table parsing
- Lavoisier element-table crosswalks
- phlogiston-to-oxygen theory concept maps
- gas-chemistry experiment reconstructions
- nomenclature dictionaries
- conservation-of-mass uncertainty examples
- closed-vessel reaction accounting
