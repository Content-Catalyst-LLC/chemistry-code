# Polymer Chemistry and Macromolecular Materials

This article folder supports the Chemistry knowledge-series article **Polymer Chemistry and Macromolecular Materials**.

The workflows use synthetic educational data to demonstrate polymer candidate screening, molar-mass distribution calculations, dispersity, thermal and mechanical property tradeoffs, degradation and circularity flags, processing-condition tracking, and full-stack scientific-computing scaffolding.

## Repository structure

- `python/` — polymer candidate screening, property-vector scoring, dispersity calculations, and provenance manifests
- `r/` — molar-mass summaries, property tradeoff rankings, and reporting
- `julia/` — molar-mass and dispersity calculations
- `fortran/` — compact polymer metric kernel
- `rust/` — polymer dataset manifest validator
- `go/` — CSV audit utility for polymer records
- `c/` — dispersity and degree-of-polymerization utility
- `cpp/` — polymer candidate classifier
- `sql/` — schema and queries for polymers, molar-mass fractions, processing, properties, and lifecycle notes
- `docs/` — methodology, reproducibility, and responsible-use notes
- `data/` — synthetic educational polymer chemistry data
- `notebooks/` — notebook scaffolds and report notes
- `outputs/` — generated tables, reports, figures, and manifests

## Responsible-use note

All data in this folder are synthetic and educational. These workflows are not validated for engineering certification, medical-device design, food-contact claims, biodegradation claims, recycling claims, or safety-critical polymer selection.
