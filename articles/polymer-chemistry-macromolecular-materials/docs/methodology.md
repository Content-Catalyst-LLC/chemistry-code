# Methodology Notes

This folder models a reproducible polymer chemistry workflow using synthetic educational data.

The workflow demonstrates:

1. polymer candidate records;
2. molar-mass fraction tables;
3. number-average and mass-average molar mass;
4. dispersity calculation;
5. processing-condition provenance;
6. property measurement tables;
7. function-target scoring;
8. lifecycle and degradation review flags;
9. full-stack scientific-computing scaffolding.

The examples are teaching workflows, not validated polymer-selection systems.
