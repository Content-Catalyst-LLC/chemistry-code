# Methodology Notes

## Purpose

This repository demonstrates reproducible computational scaffolding for intermolecular forces and condensed-matter chemistry.

## Core Relationships

Lennard-Jones potential:

U(r) = 4 epsilon [(sigma/r)^12 - (sigma/r)^6]

Coulombic interaction:

U(r) = k q1 q2 / r

Clausius-Clapeyron-style fit:

ln(P) = -Delta H_vap / (R T) + C

Radial distribution scaffold:

g(r) = local density / bulk density

Surface tension:

gamma = dG / dA

Mean-square displacement for simple three-dimensional diffusion:

MSD = 6 D t

## Scientific Caution

These examples use simplified educational assumptions. Real condensed-matter chemistry requires validated force fields, experimental reference data, careful unit conventions, uncertainty analysis, temperature and pressure specification, and expert interpretation.
