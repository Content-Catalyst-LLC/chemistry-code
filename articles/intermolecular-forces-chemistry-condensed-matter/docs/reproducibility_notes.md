# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible condensed-matter workflows should preserve:

- physical constants
- units
- interaction model
- force-field parameters
- temperature and pressure assumptions
- phase definitions
- property data sources
- fitting ranges
- simulation cell assumptions
- coordinate data
- computational scripts
- output artifacts
- software versions
- provenance records
- limitations and responsible-use notes

Future extensions could include:

- molecular dynamics input templates
- validated force-field parameter files
- vapor-liquid equilibrium datasets
- phase diagram scaffolds
- solubility parameter workflows
- surface and interface models
- polymer glass-transition examples
- radial distribution analysis from simulation trajectories
- links to public thermodynamic databases
