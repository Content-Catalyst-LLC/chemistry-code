# Responsible Use Notes

This repository is educational.

Do not use these scripts for:

- industrial process design
- safety-critical chemical operation
- electrochemical device qualification
- environmental compliance reporting
- clinical or medical decisions
- regulatory submissions
- hazardous laboratory procedures
- validated thermodynamic property prediction

Professional chemical practice requires validated data, trained personnel, calibrated instruments, documented assumptions, uncertainty analysis, safety review, and regulatory compliance where applicable.
