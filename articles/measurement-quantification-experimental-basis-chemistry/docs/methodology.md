# Methodology Notes

## Purpose

This repository demonstrates reproducible computational scaffolding for measurement and quantification in chemistry.

## Core Calculations

Amount of substance:

n = m / M

Molar concentration:

C = n / V

Dilution:

C1 V1 = C2 V2

Linear calibration:

y = mx + b

Unknown concentration from calibration:

x = (y - b) / m

Sample standard deviation:

s = sqrt(sum((xi - mean(x))^2) / (n - 1))

Relative standard deviation:

RSD = 100 * s / mean(x)

Expanded uncertainty:

U = k * uc

## Scientific Caution

These examples use simplified educational assumptions. Real laboratory chemistry requires safety review, calibration, uncertainty analysis, validated methods, proper standards, trained personnel, and domain expertise.
