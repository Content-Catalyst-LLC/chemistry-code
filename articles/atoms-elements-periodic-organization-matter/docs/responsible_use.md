# Responsible Use Notes

This repository is educational.

Do not use these scripts for:

- chemical handling decisions
- laboratory safety decisions
- regulatory compliance
- analytical validation
- toxicological assessment
- isotope-ratio interpretation without expert review
- environmental reporting
- clinical chemistry
- pharmaceutical quality control
- industrial process control

Professional chemical practice requires validated data sources, trained personnel, calibrated instruments, safety procedures, uncertainty analysis, chemical-domain expertise, and regulatory compliance where applicable.
