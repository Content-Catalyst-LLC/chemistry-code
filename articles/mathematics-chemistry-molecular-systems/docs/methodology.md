# Methodology Notes

## Purpose

This repository demonstrates reproducible computational scaffolding for mathematical chemistry.

## Core Relationships

Amount of substance:

n = m / M

Concentration:

C = n / V

pH approximation:

pH = -log10([H+])

First-order kinetics:

[A](t) = [A]0 exp(-kt)

Thermodynamics and equilibrium:

Delta G standard = -RT ln K

Molecular distance:

dij = sqrt((xi - xj)^2 + (yi - yj)^2 + (zi - zj)^2)

Combined standard uncertainty:

uc = sqrt(sum(ui^2))

## Scientific Caution

These examples use simplified educational assumptions. Real chemistry requires validated methods, clear units, calibrated instruments, uncertainty analysis, appropriate molecular models, and professional review.
