# Materials Chemistry Reproducibility Checklist

- Record composition, synthesis route, processing history, and post-treatment.
- Preserve raw and processed characterization data.
- Record measurement method, units, instrument conditions, and environment.
- Link properties to specific samples, batches, and processing records.
- Include replicate measurements and uncertainty estimates.
- Preserve simulation inputs, software versions, and convergence settings where computational data are used.
- Document application context and target performance criteria.
- Record lifecycle, toxicity, scarcity, and end-of-life assumptions.
- Distinguish screening results from validated application readiness.
