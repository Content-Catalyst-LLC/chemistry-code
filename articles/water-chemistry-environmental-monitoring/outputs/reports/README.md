# Reports

Generated reports from article-specific scripts will appear here.

Suggested commands from the article folder:

- Python: `python3 python/water_quality_screening.py`
- R: `Rscript r/water_quality_summary.R`
- Julia: `julia julia/nutrient_loads_and_oxygen.jl`
- Fortran: `cd fortran && gfortran dissolved_oxygen_saturation.f90 -o dissolved_oxygen_saturation && ./dissolved_oxygen_saturation`
- Rust: `cd rust && cargo run`
- Go: `go run go/water_quality_screening.go`
- C: `cd c && gcc ph_conductivity_screening.c -o ph_conductivity_screening && ./ph_conductivity_screening`
- C++: `cd cpp && g++ alkalinity_hardness.cpp -o alkalinity_hardness && ./alkalinity_hardness`
