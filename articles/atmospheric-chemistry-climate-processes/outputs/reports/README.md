# Reports

Generated reports from article-specific scripts will appear here.

Suggested commands from the article folder:

- Python: `python3 python/atmospheric_forcing_and_screening.py`
- R: `Rscript r/atmospheric_monitoring_summary.R`
- Julia: `julia julia/photochemical_ozone_forcing.jl`
- Fortran: `cd fortran && gfortran co2_forcing.f90 -o co2_forcing && ./co2_forcing`
- Rust: `cd rust && cargo run`
- Go: `go run go/atmospheric_screening.go`
- C: `cd c && gcc atmospheric_lifetime.c -lm -o atmospheric_lifetime && ./atmospheric_lifetime`
- C++: `cd cpp && g++ methane_oxidation_lifetime.cpp -o methane_oxidation_lifetime && ./methane_oxidation_lifetime`
