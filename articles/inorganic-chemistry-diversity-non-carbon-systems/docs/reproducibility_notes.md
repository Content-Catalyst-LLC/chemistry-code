# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible inorganic-chemistry workflows should preserve:

- compound identity
- formula and charge
- assumed oxidation states
- coordination number
- ligand definitions
- geometry
- spin-state assumptions
- crystal-field assumptions
- ionic radii sources
- material structure class
- units and descriptor definitions
- computational scripts
- output artifacts
- software versions
- provenance records
- limitations and responsible-use notes

Future extensions could include:

- full formula parsers
- ligand-field parameter fitting
- spin-state energy comparison
- d-electron counting
- Pourbaix diagram scaffolds
- phase-diagram metadata
- crystal-structure descriptors
- band-gap descriptor tables
- materials database integration
- density functional theory input templates
