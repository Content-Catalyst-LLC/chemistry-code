# Methodology Notes

## Purpose

This repository demonstrates reproducible computational scaffolding for inorganic chemistry and non-carbon-centered systems.

## Core Relationships

Charge balance:

sum_i z_i n_i = q

Oxidation-state accounting:

sum_i OS_i n_i = total charge

Coordination number:

CN = number of donor atoms directly attached to central atom

Octahedral crystal-field stabilization energy:

CFSE = n_t2g(-0.4 Delta_o) + n_eg(0.6 Delta_o)

Spin-only magnetic moment:

mu = sqrt(n(n + 2)) mu_B

Lattice-energy scaling:

U proportional to |z+ z-| / r

Goldschmidt tolerance factor:

t = (r_A + r_X) / (sqrt(2)(r_B + r_X))

## Scientific Caution

These examples use simplified educational assumptions. Real inorganic chemistry requires validated structures, spectroscopy, oxidation-state evidence, spin-state analysis, phase characterization, safety review, environmental context, uncertainty analysis, and expert interpretation.
