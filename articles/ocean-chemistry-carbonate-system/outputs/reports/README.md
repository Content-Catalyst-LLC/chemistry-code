# Reports

Generated reports from article-specific scripts will appear here.

Suggested commands from the article folder:

- Python: `python3 python/ocean_carbonate_screening.py`
- R: `Rscript r/ocean_carbonate_summary.R`
- Julia: `julia julia/carbonate_system_examples.jl`
- Fortran: `cd fortran && gfortran carbonate_speciation.f90 -o carbonate_speciation && ./carbonate_speciation`
- Rust: `cd rust && cargo run`
- Go: `go run go/ocean_carbonate_flux.go`
- C: `cd c && gcc ph_carbonate_fraction.c -lm -o ph_carbonate_fraction && ./ph_carbonate_fraction`
- C++: `cd cpp && g++ revelle_buffer_example.cpp -o revelle_buffer_example && ./revelle_buffer_example`
