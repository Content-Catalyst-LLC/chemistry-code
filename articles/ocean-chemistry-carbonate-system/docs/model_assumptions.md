# Model Assumptions

The computational examples in this folder are simplified teaching tools.

## Carbonate speciation

The carbonate-fraction workflow uses fixed illustrative values for K1 and K2. Research-grade seawater carbonate chemistry must use temperature-, salinity-, pressure-, nutrient-, and pH-scale-dependent constants.

## Saturation state

The saturation-state examples use simplified calcium and carbonate ion estimates with fixed solubility products. Real aragonite and calcite saturation calculations require pressure, temperature, salinity, calcium, carbonate, and appropriate thermodynamic constants.

## Air-sea CO2 flux

The flux proxy uses only pCO2_water - pCO2_air. Real flux calculations require gas-transfer velocity, wind speed, temperature, salinity, solubility, turbulence, sea state, ice cover, and sign convention.

## Revelle factor

The Revelle-factor example is conceptual. Real Revelle-factor calculations require full carbonate-system equilibrium treatment.

## pH scale

Marine pH must specify scale, method, and temperature basis. The synthetic data use "pH_total_scale" for teaching only.
