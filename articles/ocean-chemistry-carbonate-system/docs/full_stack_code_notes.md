# Full-Stack Computational Extensions

This folder extends the Ocean Chemistry and the Carbonate System article repository with full-stack computational examples.

## Added languages

- Julia — carbonate-system proxy modeling, saturation-state screening, and acidification scenarios
- C — low-level pH, alkalinity, DIC, carbonate-ion, and saturation kernels
- C++ — ocean-region triage and carbonate-system aggregation
- Fortran — numerical CO2 uptake and pH-shift scenario modeling
- Rust — safe systems-style carbonate chemistry screening
- Go — lightweight ocean-region and monitoring summaries
- R — statistical summaries for ocean chemistry, carbonate pressure, and acidification indicators

## Responsible-use boundary

All examples are synthetic and educational. They do not provide operational ocean monitoring, climate attribution, marine-policy determinations, fisheries management advice, legal evidence, regulatory reporting, carbon-credit validation, aquaculture siting guidance, or research-grade carbonate-system calculations.
