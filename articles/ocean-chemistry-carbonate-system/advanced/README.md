# Advanced Computational Layer: Ocean Chemistry and the Carbonate System

This folder extends the article **Ocean Chemistry and the Carbonate System** with a stronger computational layer.

## Scope

This advanced layer models simplified ocean carbonate chemistry using synthetic data. It includes carbonate speciation, dissolved inorganic carbon partitioning, aragonite and calcite saturation state, air-sea CO2 flux proxies, alkalinity-DIC buffering, Revelle-factor intuition, acidification pressure screening, uncertainty scenarios, and SQL-ready provenance structures.

## Included files

- `data/ocean_carbonate_advanced_synthetic.csv` — synthetic carbonate monitoring data
- `python/ocean_carbonate_advanced.py` — reusable advanced workflow
- `python/test_ocean_carbonate_advanced.py` — lightweight reproducibility tests
- `sql/advanced_schema.sql` — SQL provenance schema
- `sql/advanced_queries.sql` — analytical query examples
- `docs/advanced_model_notes.md` — modeling assumptions and limitations
- `docs/future_research_extensions.md` — roadmap for research-grade upgrades
- `outputs/` — generated tables, reports, and manifests

## Responsible use

This code is educational and synthetic. It is not a research-grade carbonate-system solver, ocean-acidification forecast, regulatory tool, shellfish-hatchery advisory, climate-attribution model, legal analysis, or operational monitoring system.
