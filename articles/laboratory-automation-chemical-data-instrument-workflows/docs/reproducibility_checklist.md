# Laboratory Automation Reproducibility Checklist

- Assign stable sample, container, run, method, and file identifiers.
- Preserve raw instrument files and processed outputs.
- Record instrument methods, versions, software, and acquisition parameters.
- Preserve quality-control records, calibration checks, blanks, and exceptions.
- Document sample handling, storage, preparation, and transfer steps.
- Validate data transformations and integration settings.
- Maintain audit trails for manual review, reruns, overrides, and reprocessing.
- Use checksums or equivalent controls for critical data files.
- Separate research prototypes from validated operational workflows.
