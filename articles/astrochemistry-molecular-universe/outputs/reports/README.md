# Reports

Generated reports from article-specific scripts will appear here.

Suggested commands from the article folder:

- Python: `python3 python/astrochemical_spectral_workflow.py`
- R: `Rscript r/astrochemical_survey_summary.R`
- Julia: `julia julia/reaction_network_desorption.jl`
- Fortran: `cd fortran && gfortran doppler_photon_energy.f90 -o doppler_photon_energy && ./doppler_photon_energy`
- Rust: `cd rust && cargo run`
- Go: `go run go/photochemistry_desorption.go`
- C: `cd c && gcc photon_doppler.c -lm -o photon_doppler && ./photon_doppler`
- C++: `cd cpp && g++ line_catalog_matching.cpp -o line_catalog_matching && ./line_catalog_matching`
