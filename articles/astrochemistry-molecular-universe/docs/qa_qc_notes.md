# QA/QC Notes for Astrochemical Data

Credible astrochemical interpretation requires careful spectral, laboratory, and modeling context.

## Recommended metadata

- observation identifier
- source name
- environment type
- sky coordinates
- observing date
- telescope and instrument
- spectral band
- frequency coverage
- spectral resolution
- flux calibration method
- rest-frequency database
- line uncertainty
- velocity frame
- source velocity
- beam size
- noise level
- baseline correction
- line blending notes
- candidate species
- transition assignment
- upper-state energy
- line strength
- excitation assumptions
- column density method
- molecular hydrogen reference column
- quality-control flag

## Common quality issues

- single-line molecular claims
- blended line assignments
- missing velocity-frame information
- laboratory frequency uncertainty ignored
- excitation conditions assumed without evidence
- optical depth ignored
- beam dilution ignored
- comparing incompatible column densities
- treating organic molecules as biosignatures
- overinterpreting nondetections
- applying terrestrial intuition to low-density interstellar chemistry

## Practical principle

Astrochemical data should allow another qualified reviewer to reconstruct how a molecular assignment was made, what catalog supported it, what velocity correction was applied, what transitions were considered, and what uncertainties remain.
