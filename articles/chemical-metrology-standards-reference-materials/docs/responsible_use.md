# Responsible Use Notes

This repository is educational.

Do not use these scripts for:

- laboratory safety decisions
- chemical handling
- accreditation claims
- regulatory compliance
- analytical validation
- environmental reporting
- clinical chemistry
- pharmaceutical quality control
- industrial process control

Professional chemical metrology requires validated methods, trained personnel, safety procedures, appropriate equipment, fit-for-purpose reference materials, uncertainty analysis, and regulatory or accreditation review where applicable.
