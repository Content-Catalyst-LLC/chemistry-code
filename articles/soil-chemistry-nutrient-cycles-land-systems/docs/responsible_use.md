# Responsible Use Notes

These files are educational examples for chemistry communication, computational learning, and reproducible workflow design.

They should not be used as:

- agronomic fertilizer recommendations
- lime recommendations
- contamination determinations
- legal evidence
- public-health advisories
- soil-carbon credit verification
- regulatory compliance tools
- site-remediation designs
- environmental justice screening determinations

Soil chemistry decisions can affect food production, land value, public health, ecosystems, watersheds, and climate claims. Real applications require qualified soil scientists, validated methods, appropriate standards, uncertainty analysis, and transparent communication with affected communities.

The examples intentionally use synthetic data.
