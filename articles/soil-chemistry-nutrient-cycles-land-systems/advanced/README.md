# Advanced Computational Layer: Soil Chemistry, Nutrient Cycles, and Land Systems

This folder extends the article **Soil Chemistry, Nutrient Cycles, and Land Systems** with a stronger computational layer.

## Scope

This advanced layer models simplified soil chemistry and land-system evidence using synthetic data. It includes soil organic carbon stock estimation, cation exchange capacity, base saturation, exchange acidity, nitrate leaching vulnerability, phosphorus export proxies, nutrient pressure screening, salinity and pH flags, soil carbon scenario modeling, nitrogen balance modeling, and SQL-ready provenance structures.

## Included files

- `data/soil_chemistry_advanced_synthetic.csv` — synthetic soil monitoring data
- `python/soil_chemistry_advanced.py` — reusable advanced soil chemistry workflow
- `python/test_soil_chemistry_advanced.py` — lightweight reproducibility tests
- `sql/advanced_schema.sql` — SQL provenance schema
- `sql/advanced_queries.sql` — analytical query examples
- `docs/advanced_model_notes.md` — modeling assumptions and limitations
- `docs/future_research_extensions.md` — roadmap for research-grade upgrades
- `outputs/` — generated tables, reports, and manifests

## Responsible use

This code is educational and synthetic. It is not an agronomic recommendation system, soil-carbon credit tool, contamination assessment, land valuation tool, public-health advisory, legal analysis, or regulatory reporting system.
