# Reports

Generated reports from article-specific scripts will appear here.

Suggested commands from the article folder:

- Python: `python3 python/soil_fertility_and_carbon.py`
- R: `Rscript r/soil_monitoring_summary.R`
- Julia: `julia julia/soil_carbon_and_nutrients.jl`
- Fortran: `cd fortran && gfortran soil_carbon_stock.f90 -o soil_carbon_stock && ./soil_carbon_stock`
- Rust: `cd rust && cargo run`
- Go: `go run go/soil_nutrient_balance.go`
- C: `cd c && gcc soil_ph_salinity_screening.c -o soil_ph_salinity_screening && ./soil_ph_salinity_screening`
- C++: `cd cpp && g++ cec_base_saturation.cpp -o cec_base_saturation && ./cec_base_saturation`
