# Advanced Computational Layer: Toxicology, Exposure, and Chemical Risk

This folder supports the Chemistry knowledge-series article:

**Toxicology, Exposure, and Chemical Risk**

The code is synthetic and educational. It demonstrates toxicology and exposure-science decision analytics, including chronic daily intake, route-specific exposure dose, hazard quotient, hazard index, margin of exposure, cancer-risk proxy, mixture grouping by target system, Monte Carlo uncertainty, exposure scenario modeling, SQL provenance, and responsible-use documentation.

## Responsible-use boundary

This repository does **not** determine real-world safety, public-health risk, regulatory compliance, causation, cleanup requirements, worker harm, drinking-water safety, product safety, clinical diagnosis, or legal responsibility.
