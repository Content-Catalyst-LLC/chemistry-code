# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible molecular dynamics workflows should preserve:

- starting structure
- structure-preparation steps
- topology files
- force field and version
- water model
- ion parameters
- box size and shape
- periodic boundary conditions
- minimization protocol
- equilibration protocol
- production run length
- timestep
- constraints
- thermostat
- barostat
- random seeds
- software and version
- trajectory output frequency
- analysis scripts
- statistical uncertainty
- provenance records
- responsible-use limitations

Future extensions could include:

- GROMACS input templates
- OpenMM script templates
- LAMMPS input templates
- all-atom trajectory-analysis examples
- RMSD and RMSF scaffolds
- hydrogen-bond analysis
- radial distribution functions from coordinates
- diffusion analysis with block averaging
- free-energy surface scaffolds
- enhanced-sampling metadata
- replica-analysis workflows
