# Advanced Model Notes: Geochemistry and the Chemical History of Earth

This advanced layer is designed to make the GitHub repository more credible as a computational science companion to the article.

## What this layer adds

- major-oxide molar conversion
- weight-based and molar Chemical Index of Alteration
- simplified CaO* correction proxy
- trace-element ratios
- isotope delta notation
- simplified parent-daughter radiometric dating
- rare-earth-element normalization
- light-to-heavy REE enrichment proxy
- redox archive index
- geochemical archive index
- radiogenic decay series
- isotope mixing series
- weathering trajectory series
- SQL-ready provenance structure
- lightweight tests

## Important simplifications

The workflow is intentionally educational. It does not include:

- full mineralogical correction of CaO*
- carbonate/phosphate petrographic confirmation
- certified reference material QA/QC
- analytical uncertainty propagation
- true concordia/discordia geochronology
- isotope system closure-temperature analysis
- rare-earth standard selection workflows
- real Eu anomaly calculation
- stratigraphic context
- petrographic validation
- alteration screening by microscopy
- professional geochemical classification diagrams

## Interpretation

The output indices are screening and teaching indicators. They help explain how geochemistry turns chemical measurements into evidence, but they are not scientific findings.

## Responsible-use boundary

Do not use these outputs for mining, exploration, environmental contamination assessment, geochronology, legal evidence, public communication of actual field samples, resource estimation, investment decisions, or regulatory reporting.
