# Advanced Computational Layer: Geochemistry and the Chemical History of Earth

This folder extends the article **Geochemistry and the Chemical History of Earth** with a stronger computational layer.

## Scope

This advanced layer models simplified geochemical evidence using synthetic data. It includes major-oxide geochemistry, molar Chemical Index of Alteration, trace-element ratios, isotope delta notation, simplified radiometric dating, rare-earth-element normalization, redox archive screening, weathering trajectories, two-component isotope mixing, and SQL-ready provenance structures.

## Included files

- `data/geochemistry_advanced_synthetic.csv` — synthetic geochemical sample data
- `python/geochemistry_advanced.py` — reusable advanced geochemistry workflow
- `python/test_geochemistry_advanced.py` — lightweight reproducibility tests
- `sql/advanced_schema.sql` — SQL provenance schema
- `sql/advanced_queries.sql` — analytical query examples
- `docs/advanced_model_notes.md` — modeling assumptions and limitations
- `docs/future_research_extensions.md` — roadmap for research-grade upgrades
- `outputs/` — generated tables, reports, and manifests

## Responsible use

This code is educational and synthetic. It is not a professional geochemical interpretation, mining assessment, geochronology report, contamination assessment, legal analysis, resource estimate, or regulatory tool.
