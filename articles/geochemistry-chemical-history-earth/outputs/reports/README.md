# Reports

Generated reports from article-specific scripts will appear here.

Suggested commands from the article folder:

- Python: `python3 python/geochemistry_indices_and_ages.py`
- R: `Rscript r/geochemical_summary.R`
- Julia: `julia julia/geochemical_ratios.jl`
- Fortran: `cd fortran && gfortran radiometric_age.f90 -o radiometric_age && ./radiometric_age`
- Rust: `cd rust && cargo run`
- Go: `go run go/geochemical_age_ratios.go`
- C: `cd c && gcc isotope_delta.c -o isotope_delta && ./isotope_delta`
- C++: `cd cpp && g++ ree_normalization.cpp -o ree_normalization && ./ree_normalization`
