# QA/QC Notes for Geochemical Data

Credible geochemical interpretation requires strong analytical and geological context.

## Recommended metadata

- sample identifier
- field location
- coordinate system
- sample date
- collector
- material type
- rock type
- stratigraphic or geologic context
- sample preparation method
- digestion method
- analytical instrument
- laboratory identifier
- reference materials
- blanks
- duplicates
- detection limits
- uncertainty
- calibration method
- isotope standard
- mineral phase analyzed
- petrographic context
- alteration notes
- QA/QC flags

## Common quality issues

- interpreting bulk chemistry without mineralogy
- ignoring alteration or weathering
- comparing incompatible analytical methods
- using oxide weight percentages where molar proportions are required
- failing to correct CaO* in CIA calculations
- ignoring detection limits for trace elements
- treating isotope ratios without standards or uncertainty
- applying radiometric equations without closed-system checks
- treating a single proxy as definitive
- ignoring field relationships and stratigraphy

## Practical principle

Geochemical data should allow another qualified reviewer to reconstruct where the sample came from, what material was analyzed, how it was prepared, what method produced the result, what uncertainty applies, and what interpretation is justified.
