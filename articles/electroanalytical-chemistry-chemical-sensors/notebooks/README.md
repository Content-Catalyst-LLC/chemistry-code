# Notebook Notes

Suggested notebook scaffolds:

1. `electrochemical_sensor_audit.ipynb`
   - imports synthetic calibration, unknown, drift, potentiometry, voltammetry, interference, and metadata tables;
   - calculates sensitivity and detection limit;
   - estimates unknown concentrations;
   - flags interferences;
   - writes reproducibility outputs.

2. `electrochemical_sensor_report.qmd`
   - renders a reproducible teaching report;
   - links figures and tables to generated outputs;
   - documents limitations and responsible-use boundaries.

Notebook files should be executed from a clean kernel to avoid hidden-state errors.
