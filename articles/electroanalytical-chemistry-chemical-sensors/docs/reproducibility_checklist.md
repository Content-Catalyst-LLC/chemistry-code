# Electroanalytical Reproducibility Checklist

- Record electrode material, geometry, pretreatment, and modification chemistry.
- Record reference electrode type, electrolyte composition, pH, and temperature.
- Preserve raw electrochemical data files.
- Document baseline subtraction, smoothing, peak picking, drift correction, and calibration fitting.
- Report sensitivity, linear range, detection limit, response time, repeatability, and reproducibility.
- Test interferents and matrix effects.
- Track electrode-to-electrode variability and sensor lifetime.
- Distinguish screening use from validated analytical use.
