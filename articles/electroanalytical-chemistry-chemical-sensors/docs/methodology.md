# Methodology Notes

This folder models a reproducible electroanalytical chemistry workflow using synthetic educational data.

The workflow demonstrates:

1. amperometric calibration;
2. detection-limit estimation;
3. unknown concentration estimation;
4. potentiometric Nernst-like response;
5. voltammetric peak summaries;
6. sensor drift assessment;
7. interference flagging;
8. electrochemical method metadata;
9. full-stack scientific-computing scaffolding.

The examples are teaching workflows, not validated analytical methods.
