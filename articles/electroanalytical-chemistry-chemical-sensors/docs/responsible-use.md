# Responsible Use Notes

Electrochemical sensors are widely used in clinical, environmental, food-safety, industrial, occupational, corrosion, and field-monitoring contexts. The examples in this folder are not validated for those purposes.

## Do not use this folder for

- clinical diagnosis or treatment;
- environmental compliance;
- forensic identification;
- industrial process release;
- occupational exposure decisions;
- safety-critical chemical detection;
- regulatory reporting.

## Recommended real-world extensions

- preserve raw current, potential, time, and impedance data;
- document electrode material, reference electrode, electrolyte, pH, temperature, and surface preparation;
- validate calibration range, sensitivity, detection limit, response time, and drift;
- test matrix effects and realistic interferences;
- compare sensor results with reference methods where decisions are consequential;
- report uncertainty and limitations honestly;
- distinguish prototype performance from validated measurement capability.
