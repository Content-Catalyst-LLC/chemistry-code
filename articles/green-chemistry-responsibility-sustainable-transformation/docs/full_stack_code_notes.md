# Full-Stack Computational Extensions

This folder extends the Green Chemistry, Responsibility, and Sustainable Transformation article repository with full-stack computational examples.

## Added languages

- Python — full green chemistry screening workflow and reports
- SQL — provenance schema and route-query examples
- Julia — route scoring and green chemistry metric modeling
- C — atom economy, E-factor, PMI, and hazard-weighted kernels
- C++ — route triage and chemistry-class aggregation
- Fortran — process mass and waste-reduction scenario modeling
- Rust — safe systems-style route screening
- Go — lightweight route-class summaries
- R — statistical summaries for routes and chemistry classes

## Responsible-use boundary

All examples are synthetic and educational. They do not certify green chemistry claims, validate industrial processes, determine regulatory compliance, replace life-cycle assessment, approve solvents, determine worker safety, evaluate real toxicity, support product claims, or authorize environmental marketing statements.
