# Methodology Notes

## Purpose

This repository demonstrates reproducible computational scaffolding for acid-base chemistry and proton-transfer systems.

## Core Relationships

Water autoionization:

Kw = [H3O+][OH-]

pH:

pH = -log10([H3O+])

pOH:

pOH = -log10([OH-])

Acid dissociation:

Ka = [H3O+][A-] / [HA]

Base dissociation:

Kb = [BH+][OH-] / [B]

pKa:

pKa = -log10(Ka)

Conjugate relationship:

Ka Kb = Kw

Henderson-Hasselbalch equation:

pH = pKa + log10([A-]/[HA])

Weak acid approximation:

[H3O+] approx sqrt(Ka C)

Charge balance:

sum(z_i c_i) = 0

## Scientific Caution

These examples use simplified educational assumptions. Real acid-base work requires temperature control, calibrated pH measurement, activity corrections where appropriate, ionic strength awareness, concentration and purity documentation, uncertainty analysis, and expert interpretation.
