# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible acid-base workflows should preserve:

- acid/base identities
- concentrations and units
- Ka, Kb, pKa, pKb values and sources
- temperature
- solvent system
- activity assumptions
- ionic strength assumptions
- mass balance assumptions
- charge balance assumptions
- titration stoichiometry
- endpoint/equivalence definitions
- computational scripts
- output artifacts
- software versions
- provenance records
- limitations and responsible-use notes

Future extensions could include:

- charge-balance pH solving
- activity coefficient corrections
- polyprotic acid nonlinear solvers
- carbonate system modeling
- amino acid charge-state workflows
- buffer capacity calculations
- Gran plot analysis
- pH meter calibration records
- uncertainty propagation
- environmental speciation models
