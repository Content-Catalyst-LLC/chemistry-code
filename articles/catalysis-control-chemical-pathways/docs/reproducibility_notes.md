# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible catalysis workflows should preserve:

- catalyst identity and loading
- substrate concentrations
- temperature and pressure
- solvent or phase
- reaction time
- product quantification method
- catalyst state and pretreatment
- turnover metrics
- kinetic model assumptions
- adsorption model assumptions
- deactivation assumptions
- mass-transfer checks
- computational scripts
- output artifacts
- software versions
- provenance records
- limitations and responsible-use notes

Future extensions could include:

- nonlinear kinetic fitting
- catalyst deactivation parameter estimation
- microkinetic network models
- adsorption competition
- reaction-order estimation
- uncertainty propagation
- kinetic isotope effects
- enzyme inhibition models
- heterogeneous reactor models
- catalyst screening dashboards
