# Catalysis and the Control of Chemical Pathways

This article repository supports the Chemistry knowledge-series article:

Catalysis and the Control of Chemical Pathways

The repository provides reproducible computational scaffolding for catalysis, including barrier lowering, Arrhenius rate enhancement, turnover number, turnover frequency, catalytic activity, Michaelis-Menten saturation, Langmuir adsorption, Langmuir-Hinshelwood surface-rate scaffolds, catalytic-cycle ODE models, deactivation examples, SQL-backed metadata, notebook scaffolding, and cross-language examples in Python, R, Julia, Fortran, Rust, Go, C, and C++.

No university affiliation is implied by this repository. The design goal is advanced reproducible computational rigor: clear assumptions, transparent calculations, synthetic educational data, provenance records, responsible-use notes, and reusable scientific workflow structure.

## Scientific Focus

The workflows demonstrate:

1. Arrhenius barrier-lowering rate enhancement.
2. Turnover number and turnover frequency.
3. Catalytic activity in mol s^-1.
4. Michaelis-Menten catalytic saturation.
5. Langmuir adsorption coverage.
6. Langmuir-Hinshelwood surface-rate scaffolds.
7. Catalytic cycle ODE models.
8. Catalyst deactivation examples.
9. SQL-backed catalytic metadata.
10. Cross-language numerical reproducibility.

These examples are educational and methodological scaffolds. They are not validated catalyst design tools, industrial reactor models, enzyme assay tools, chemical safety systems, regulatory tools, or substitutes for professional chemical practice.
