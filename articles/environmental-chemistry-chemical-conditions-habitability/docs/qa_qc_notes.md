# QA/QC Notes for Environmental Chemistry Data

A credible environmental chemistry workflow should preserve enough information to make data auditable.

## Recommended fields

- sample identifier
- site identifier
- date and time
- latitude and longitude
- environmental medium
- analyte
- concentration
- unit
- detection limit
- reporting limit
- analytical method
- sample preservation method
- laboratory identifier
- calibration status
- blank results
- duplicate results
- spike recovery
- qualifier flags
- benchmark source
- data-review status

## Common quality issues

- unit mismatch
- censored values below detection limits
- contaminated blanks
- matrix interference
- sample holding-time exceedance
- inconsistent filtration status
- missing coordinates
- missing benchmark context
- inappropriate comparison between total and dissolved concentrations
- treating screening exceedance as final risk determination

## Practical principle

Environmental data should be structured so another qualified reviewer can reconstruct how the result was produced, what it means, and what it does not mean.
