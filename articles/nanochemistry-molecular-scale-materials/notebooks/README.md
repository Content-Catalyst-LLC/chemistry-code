# Notebook Notes

Suggested notebook scaffolds:

1. `nanochemistry_audit.ipynb`
   - imports synthetic nanomaterial, replicate, ligand, optical, stability, and lifecycle tables;
   - calculates surface-area-to-volume and diffusion estimates;
   - ranks candidates;
   - flags stability and responsible-design review items;
   - writes reproducibility outputs.

2. `nanochemistry_report.qmd`
   - renders a reproducible teaching report;
   - links figures and tables to generated outputs;
   - documents limitations and responsible-use boundaries.

Notebook files should be executed from a clean kernel to avoid hidden-state errors.
