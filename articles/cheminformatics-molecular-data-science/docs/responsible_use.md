# Responsible Use Notes

This repository is educational.

Do not use these scripts for:

- clinical decisions
- drug safety decisions
- regulatory submissions
- toxicity classification
- controlled-substance design
- hazardous chemical design
- chemical weapons work
- environmental compliance
- production compound prioritization without expert review

Professional cheminformatics requires validated chemical structures, carefully curated data, domain-specific interpretation, uncertainty analysis, applicability-domain checks, and ethical review where relevant.
