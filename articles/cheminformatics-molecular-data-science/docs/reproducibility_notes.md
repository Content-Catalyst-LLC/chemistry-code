# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible cheminformatics workflows should preserve:

- data source
- download date
- database version
- license
- original structure string
- standardized structure string
- identifier mapping
- salt handling policy
- tautomer handling policy
- stereochemistry policy
- descriptor definitions
- fingerprint parameters
- assay filtering rules
- unit conversion rules
- train/test split rules
- random seeds
- model settings
- evaluation metrics
- applicability-domain checks
- uncertainty estimates
- provenance records
- responsible-use limitations

Future extensions could include:

- RDKit workflows
- PubChem identifier mapping
- ChEMBL bioactivity import
- scaffold extraction
- Bemis-Murcko scaffold split
- QSAR benchmarking
- chemical-space visualization
- reaction-informatics metadata
- molecular knowledge graph scaffolds
- model cards for molecular predictors
