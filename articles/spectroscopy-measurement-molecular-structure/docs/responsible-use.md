# Responsible Use Notes

Spectroscopic workflows can inform important chemical, environmental, medical, industrial, forensic, and materials decisions. The examples in this folder are not validated for those purposes.

## Do not use this folder for

- clinical diagnosis or treatment;
- forensic identification;
- regulatory compliance;
- environmental enforcement;
- pharmaceutical release;
- industrial quality certification;
- hazardous-material identification;
- safety-critical chemical decisions.

## Recommended real-world extensions

- preserve raw instrument files;
- document sample preparation and chain of custody where relevant;
- validate spectral-processing methods;
- report calibration range, detection limits, and uncertainty;
- use certified reference materials where appropriate;
- compare unknowns against curated reference spectra;
- confirm important structural claims using orthogonal methods.
