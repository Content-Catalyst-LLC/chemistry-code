# Methodology Notes

This folder models a reproducible spectroscopic workflow using synthetic educational data.

The workflow demonstrates:

1. IR peak tables and approximate educational functional-region assignment.
2. UV-visible calibration using synthetic standards and replicate unknown measurements.
3. NMR signal summaries for chemical-shift and integration reasoning.
4. Photon-energy conversion from wavelength and wavenumber.
5. Provenance documentation for spectral methods, instruments, and samples.
6. Full-stack scientific-computing scaffolding across multiple languages.

The assignments in this folder are teaching examples, not validated structure determinations.
