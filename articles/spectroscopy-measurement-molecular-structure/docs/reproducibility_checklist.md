# Spectroscopy Reproducibility Checklist

- Preserve raw spectra or instrument-export files.
- Record method, instrument ID, acquisition date, operator, and sample ID.
- Record solvent, matrix, concentration, temperature, and preparation history.
- Document spectral range, resolution, path length, scan number, and baseline correction.
- Preserve calibration records and reference materials.
- Separate automated peak suggestions from expert interpretation.
- Store processed data and figures with source provenance.
- Report uncertainty, limitations, and alternative interpretations.
