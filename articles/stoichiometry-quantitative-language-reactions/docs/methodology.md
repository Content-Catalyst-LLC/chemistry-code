# Methodology Notes

## Purpose

This repository demonstrates reproducible computational scaffolding for stoichiometry and quantitative reaction reasoning.

## Core Relationships

Amount of substance and entities:

N = n N_A

Mass and molar mass:

n = m / M

Stoichiometric ratio:

n_A / a = n_B / b

Limiting reaction extent:

xi_max = min(n_i / |nu_i|)

Percent yield:

percent_yield = actual_yield / theoretical_yield * 100

Amount concentration:

C = n / V

Dilution:

C1 V1 = C2 V2

Titration equivalence:

C_A V_A / a = C_B V_B / b

Ideal gas relationship:

P V = n R T

Reaction extent:

n_i = n_i0 + nu_i xi

## Scientific Caution

These examples use simplified educational assumptions. Real chemical work requires validated reagents, balanced equations, clear units, purity corrections, uncertainty analysis, safety review, and professional interpretation.
