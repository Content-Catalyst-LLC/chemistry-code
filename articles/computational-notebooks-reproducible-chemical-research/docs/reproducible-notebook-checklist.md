# Reproducible Chemical Notebook Checklist

Use this checklist when preparing computational notebooks for chemical research, teaching, or internal review.

## Scientific context

- State the chemical question.
- Identify molecules, materials, samples, standards, or reactions.
- Define units explicitly.
- Describe instrument methods or data sources.
- Distinguish raw data from processed data.

## Computational context

- Record language versions.
- Record package versions.
- Use an environment file where possible.
- Set random seeds for stochastic workflows.
- Keep code cells executable from a clean session.
- Avoid hidden state and out-of-order execution.

## Data and provenance

- Preserve raw files where possible.
- Store derived outputs in named folders.
- Generate checksums for important files.
- Record instrument identifiers and method settings.
- Record analyst, date, and notebook version.
- Link figures and tables to their source data.

## Interpretation

- State model assumptions.
- Include uncertainty estimates where relevant.
- Explain exclusions or transformations.
- Separate exploratory analysis from confirmed claims.
- Document limitations and known failure modes.
