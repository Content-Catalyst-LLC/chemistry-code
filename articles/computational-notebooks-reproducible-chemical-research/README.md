# Computational Notebooks and Reproducible Chemical Research

This article folder supports the Chemistry knowledge-series article **Computational Notebooks and Reproducible Chemical Research**.

The code examples use synthetic educational chemical-measurement data to demonstrate reproducible notebook practices, calibration workflows, provenance manifests, environment checks, and multi-language scientific-computing scaffolding.

## Repository structure

- `python/` — notebook audit and calibration provenance workflow
- `r/` — statistical reproducibility summaries for chemical measurements
- `julia/` — environment-sensitivity and measurement-summary workflow
- `fortran/` — lightweight checksum utility for data-integrity checks
- `rust/` — command-line manifest validator
- `go/` — CSV audit utility for notebook execution records
- `c/` — row-count and checksum utility for raw-data auditing
- `cpp/` — provenance graph extraction from synthetic run metadata
- `sql/` — schema and queries for notebook provenance and reproducibility records
- `docs/` — responsible-use notes and reproducible-notebook checklist
- `data/` — synthetic educational data
- `notebooks/` — notebook implementation notes
- `outputs/` — generated reports, figures, tables, and manifests

## Responsible-use note

The dataset in this folder is synthetic. It is designed for educational demonstration and should not be used for regulatory, clinical, environmental-compliance, industrial-quality, or safety-critical chemical decisions.
