# Responsible Use Notes

This repository is educational.

Do not use these scripts for:

- chemical safety decisions
- industrial process control
- reactor operation
- pharmaceutical manufacturing
- environmental compliance reporting
- battery safety analysis
- validated calorimetry
- thermodynamic database publication
- regulatory submissions
- hazardous chemical design

Professional chemical practice requires validated methods, trained personnel, documented assumptions, evaluated reference data, uncertainty analysis, safety review, and regulatory compliance where applicable.
