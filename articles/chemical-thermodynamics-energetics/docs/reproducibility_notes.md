# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible thermodynamic workflows should preserve:

- reaction equation
- balanced coefficients
- physical states
- standard-state conventions
- temperature and pressure
- heat capacity assumptions
- calorimeter assumptions
- formation enthalpy values and sources
- entropy values and sources
- reaction quotient assumptions
- equilibrium constant units/conventions
- activity versus concentration assumptions
- computational scripts
- output artifacts
- software versions
- provenance records
- limitations and responsible-use notes

Future extensions could include:

- evaluated thermodynamic database ingestion
- NASA polynomial heat capacity examples
- activity coefficient corrections
- phase diagram scaffolds
- electrochemical free-energy workflows
- reaction-coupling networks
- uncertainty propagation for calorimetry
- process heat-integration examples
- equilibrium composition solvers
