# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible equilibrium workflows should preserve:

- balanced reaction equation
- stoichiometric coefficients
- equilibrium constant definition
- temperature and pressure
- standard-state convention
- activity or concentration basis
- phase assumptions
- mass-balance constraints
- charge-balance constraints when relevant
- activity coefficient model
- solver assumptions
- computational scripts
- output artifacts
- software versions
- provenance records
- limitations and responsible-use notes

Future extensions could include:

- multi-equilibrium speciation solvers
- charge-balance pH solving
- carbonate-system scaffolds
- metal-complex formation equilibria
- Debye-Huckel and Davies corrections
- fugacity corrections for gas equilibria
- phase-equilibrium modules
- uncertainty propagation
- thermodynamic database ingestion
