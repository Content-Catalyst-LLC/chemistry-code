# Full-Stack Computational Extensions

This folder extends the advanced Medicinal Chemistry and Drug Discovery repository beyond Python and SQL.

## Added languages

- Julia — multiparameter optimization and potency/selectivity tradeoff modeling
- C — low-level potency, LLE, and liability scoring kernels
- C++ — Pareto frontier and candidate triage modeling
- Fortran — numerical PK/exposure intuition and half-life modeling
- Rust — safe systems-style candidate scoring and decision logic
- Go — lightweight project analytics and CLI-style summaries
- R — project-level discovery analytics and statistical summaries

## Responsible-use boundary

All examples are synthetic and educational. They do not provide synthesis routes, clinical recommendations, patient-treatment guidance, dosing advice, toxicology clearance, regulatory conclusions, controlled-substance design, real compound recommendations, or substitutes for professional medicinal chemistry, pharmacology, toxicology, clinical, legal, or regulatory review.
