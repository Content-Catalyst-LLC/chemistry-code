# Advanced Computational Layer: Medicinal Chemistry and Drug Discovery

This folder extends the article **Medicinal Chemistry and Drug Discovery** with a professional-facing computational layer for medicinal chemists, cheminformatics teams, discovery scientists, translational researchers, and drug-discovery strategists.

## Scope

This advanced layer models simplified medicinal chemistry decision evidence using synthetic compound data. It includes potency conversion, pIC50, ligand efficiency, lipophilic ligand efficiency, ligand-lipophilicity tradeoffs, selectivity windows, Lipinski and Veber-style property checks, ADMET screening, hERG and CYP risk flags, solubility and permeability triage, microsomal stability, pharmacokinetic intuition, multiparameter optimization, Pareto frontier analysis, Monte Carlo decision uncertainty, assay progression logic, and SQL-ready provenance structures.

## Included files

- `data/medicinal_chemistry_advanced_synthetic.csv` — synthetic compound and assay data
- `python/medicinal_chemistry_advanced.py` — reusable advanced medicinal chemistry workflow
- `python/test_medicinal_chemistry_advanced.py` — lightweight reproducibility tests
- `sql/advanced_schema.sql` — SQL provenance schema
- `sql/advanced_queries.sql` — analytical query examples
- `docs/advanced_model_notes.md` — professional modeling assumptions and limitations
- `docs/future_research_extensions.md` — roadmap for research-grade upgrades
- `outputs/` — generated indicators, scenario outputs, reports, and manifests

## Responsible use

This code is educational and synthetic. It is not a clinical decision system, patient-treatment tool, dosing model, regulatory submission, toxicology determination, synthesis protocol, controlled-substance design workflow, biological-weapons workflow, or substitute for qualified medicinal chemistry, pharmacology, toxicology, clinical, legal, or regulatory review.
