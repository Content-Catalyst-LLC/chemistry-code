# Reproducibility Notes

All datasets are synthetic or simplified educational datasets.

Reproducible molecular-geometry workflows should preserve:

- coordinate source
- atom identifiers
- element identities
- units
- molecular graph assumptions
- bond definitions
- geometry conventions
- conformer labels
- symmetry-operation assumptions
- computational scripts
- output artifacts
- software versions
- provenance records
- limitations and responsible-use notes

Future extensions could include:

- XYZ/SDF/MOL2/PDB parsers
- conformer libraries
- Kabsch alignment for RMSD
- point-group detection scaffolding
- crystal unit-cell metadata
- symmetry operation tables
- molecular surface calculations
- quantum-chemistry geometry optimization templates
- structural-database validation workflows
