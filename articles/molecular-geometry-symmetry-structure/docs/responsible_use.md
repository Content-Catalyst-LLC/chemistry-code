# Responsible Use Notes

This repository is educational.

Do not use these scripts for:

- laboratory safety decisions
- chemical handling decisions
- molecular design decisions without expert review
- drug discovery decisions
- environmental reporting
- crystallographic validation
- regulatory compliance
- pharmaceutical quality control
- industrial process control
- materials qualification

Professional chemical practice requires validated data sources, trained personnel, documented assumptions, appropriate software, uncertainty analysis, safety procedures, and regulatory compliance where applicable.
