# Methodology Notes

## Purpose

This repository demonstrates reproducible computational scaffolding for introductory electronic-structure chemistry.

## Core Relationships

Photon energy:

E = h nu

Wavelength and frequency:

c = lambda nu

Hydrogen-like energy level:

E_n = -13.6 eV / n^2

Probability density:

P proportional to |psi|^2

Subshell orbital count:

orbitals = 2l + 1

Subshell electron capacity:

maximum electrons = 2(2l + 1)

Approximate effective nuclear charge:

Z_eff = Z - S

Matrix eigenvalue form:

H c = E c

## Scientific Caution

These examples use simplified educational assumptions. Real electronic-structure research requires validated software, careful physical models, basis-set choices, convergence checks, electron-correlation treatment, relativistic treatment where appropriate, uncertainty analysis, and expert interpretation.
